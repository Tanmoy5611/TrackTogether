CO2 calculation model

Baseline vs actual emissions

[[#{html.the.baseline.assumes.that.each.participant.travels.alone.by.car}]]

  • Baseline emissions = distance x participants x 0.120 kg CO2/km
  • [[#{html.car}]] actual emissions = distance x participants x 0.120 kg CO2/km
  • [[#{html.carpool}]] actual emissions = distance x 0.120 kg CO2/km for the shared car
  • [[#{html.public.transport}]] actual emissions = distance x participants x 0.040 kg CO2/km
  • [[#{html.bike}]] and walking actual emissions = 0 kg CO2/km
  • CO2 savings = baseline emissions - actual emissions
